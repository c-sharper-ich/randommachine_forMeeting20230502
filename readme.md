# Random Viewer for Meeting 20230502

これは部活用の自己紹介の順番を決める用のアプリ

`MainWindow.xaml` , `MainWindow.xaml.cs` に基本実装は書かれている。

画面左側が, 表示される内容の設定。

"Next"ボタンで, 次の文字列を指定  
"Set"ボタンで, 左側中央部に改行区切りで入力した文字列をリスト化して, リセット。これ以降"Next"を押せば, リスト化された内容がランダムに順番に表示される。  
"Normal" <=> "Top" 常にウィンドウを手前に表示するか否か。
